## Description

Mind Reader is a Python module that lets you play a fun mind reading game interactively through your command line interface. It uses `pyttsx3` for text-to-speech capabilities and `random` module for generating random numbers to add to your mental calculations.

**Caution:** This module is designed solely for entertainment purposes. Please do not modify the code to avoid potential system instability.

## Features

- Interactive mind reading game.
- Uses text-to-speech to guide users through each step.
- Generates random numbers to add to user's calculations.
